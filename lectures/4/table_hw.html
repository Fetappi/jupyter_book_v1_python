Таблица с заданиями#

Общие требования ко всем заданиям:

  • Вам необходимо реализовать работу списка со словарем.

  • При реализации программы запрещено пользоваться классами!!!

  • Список содержит словари, каждый словарь — отдельный объект учёта (книга, расход, студент и т. п.).

  • Первая функция всегда добавляет новый элемент в список.

  • Остальные три функции выполняют: анализ данных, изменение полей в словарях, удаление элементов по критерию.

  • Функции должны проверять существование элемента перед изменением/удалением и выводить сообщения об ошибках.

  • Ввод/вывод через print и input допускается для интерактивности.

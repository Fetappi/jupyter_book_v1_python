Допольнительрые вопросы#

Задание 1. Учёт книг в домашней библиотеке#

  1. Изменить add_book(), добавив необязательный параметр genre=None. В словарь книги добавить поле genre.

  2. Написать функцию get_books_from_year(year), возвращающую список книг заданного года издания.

Задание 2. Планирование бюджета на месяц#

  1. Изменить add_expense(), добавив необязательный параметр description='' (описание расхода). Добавить поле в словарь расхода.

  2. Написать функцию expenses_by_month(year, month), возвращающую сумму расходов за указанный месяц (формат даты: 'YYYY-MM-DD').

Задание 3. Система учёта оценок студентов#

  1. Изменить add_grade(), добавив необязательный параметр subject='Общий' (предмет). Добавить поле subject в словарь.

  2. Написать функцию avg_grade_by_subject(subject), возвращающую средний балл по указанному предмету.

Задание 4. Трекер привычек#

  1. Изменить add_habit(), добавив параметр target_days_per_week=5 (цель — сколько дней в неделю выполнять). Добавить поле в словарь.

  2. Написать функцию habits_below_target(), возвращающую привычки, выполненные меньше целевого количества дней за последнюю неделю.

Задание 5. Список фильмов для просмотра#

  1. Изменить add_movie(), добавив параметр director=None. Добавить поле director в словарь.

  2. Написать функцию movies_by_director(director), возвращающую список фильмов указанного режиссёра.

Задание 6. Планирование тренировок#

  1. Изменить log_workout(), добавив параметр notes='' (примечания к тренировке). Добавить поле в словарь.

  2. Написать функцию workouts_by_intensity(intensity), возвращающую тренировки заданной интенсивности ('низкая', 'средняя', 'высокая').

Задание 7. Учёт расходов в поездке#

  1. Изменить add_trip_expense(), добавив параметр currency='RUB'. Добавить поле в словарь.

  2. Написать функцию convert_to_currency(target_currency, rates), конвертирующую все расходы в указанную валюту (по словарю курсов rates).

Задание 8. Дневник питания#

  1. Изменить add_food(), добавив параметр protein=0 (белки в г на 100 г). Добавить поле в словарь.

  2. Написать функцию daily_protein(date), возвращающую общее количество белков за день.

Задание 9. Учёт задач проекта#

  1. Изменить add_task(), добавив параметр assigned_to=None (исполнитель). Добавить поле в словарь.

  2. Написать функцию tasks_by_assignee(assignee), возвращающую задачи, назначенные указанному исполнителю.

Задание 10. Каталог книг в библиотеке#

  1. Изменить add_book_to_library(), добавив параметр isbn=None. Добавить поле isbn в словарь.

  2. Написать функцию find_by_isbn(isbn), возвращающую книгу по ISBN или None, если не найдена.

Задание 11. Учёт сотрудников компании#

  1. Изменить add_employee(), добавив параметр email=None. Добавить поле в словарь.

  2. Написать функцию employees_by_email_domain(domain), возвращающую сотрудников с почтой на указанном домене (например, 'company.com').

Задание 12. Каталог автомобилей#

  1. Изменить add_car(), добавив параметр fuel_type='бензин'. Добавить поле в словарь.

  2. Написать функцию cars_by_fuel_type(fuel_type), возвращающую автомобили заданного типа топлива.

Задание 13. Журнал погоды#

  1. Изменить record_weather(), добавив параметр humidity=0 (влажность %). Добавить поле в словарь.

  2. Написать функцию high_humidity_days(threshold), возвращающую даты с влажностью выше порога.

Задание 14. Список рецептов#

  1. Изменить add_recipe(), добавив параметр calories=0 (калорийность блюда). Добавить поле в словарь.

  2. Написать функцию low_calorie_recipes(max_calories), возвращающую рецепты с калорийностью не выше указанного значения.

Задание 15. Учёт задач на день#

  1. Изменить add_task(), добавив параметр due_date=None (дата выполнения). Добавить поле в словарь.

  2. Написать функцию overdue_tasks(), возвращающую задачи с просроченной датой выполнения (если due_date задана и прошла).

Задание 16. Каталог музыки#

  1. Изменить add_track(), добавив параметр album=None. Добавить поле в словарь.

  2. Написать функцию tracks_by_album(album), возвращающую треки из указанного альбома.

Задание 17. Журнал тренировок#

  1. Изменить log_workout(), добавив параметр heart_rate=0 (пульс во время тренировки). Добавить поле в словарь.

  2. Написать функцию avg_heart_rate_by_exercise(exercise), возвращающую средний пульс для указанного упражнения.

Задание 18. Учёт подписок#

  1. Изменить add_subscription(), добавив параметр payment_method='карта'. Добавить поле в словарь.

  2. Написать функцию subscriptions_by_method(method), возвращающую подписки с указанным способом оплаты.

Задание 19. Список мест для посещения#

  1. Изменить add_place(), добавив параметр cost=0 (примерная стоимость посещения). Добавить поле в словарь.

  2. Написать функцию budget_places(max_cost), возвращающую места с ценой не выше указанной.

Задание 20. Учёт расходов на транспорт#

  1. Изменить add_transport_expense(), добавив параметр distance=0 (пройденное расстояние в км). Добавить поле в словарь.

  2. Написать функцию cost_per_km(), возвращающую среднюю стоимость километра пути по всем записям.

Задание 21. Список целей на год#

  1. Изменить add_goal(), добавив параметр category='личная'. Добавить поле в словарь.

  2. Написать функцию goals_by_category(category), возвращающую цели указанной категории.

Задание 22. Каталог фильмов#

  1. Изменить add_movie(), добавив параметр actors=[] (список актёров). Добавить поле в словарь.

  2. Написать функцию movies_with_actor(actor), возвращающую фильмы с участием указанного актёра.

Задание 23. Учёт домашних животных#

  1. Изменить add_pet(), добавив параметр breed=None (порода). Добавить поле в словарь.

  2. Написать функцию pets_by_breed(breed), возвращающую питомцев указанной породы.

Задание 24. Каталог растений#

  1. Изменить add_plant(), добавив параметр toxicity='нетоксичное'. Добавить поле в словарь.

  2. Написать функцию toxic_plants(), возвращающую растения с пометкой 'токсичное'.

Задание 25. Учёт оборудования в лаборатории#

  1. Изменить add_equipment(), добавив параметр purchase_date=None. Добавить поле в словарь.

  2. Написать функцию equipment_age(), возвращающую словарь {название: количество лет} для всего оборудования.

Задание 26. Список контактов#

  1. Изменить add_contact(), добавив параметр birthday=None (дата рождения). Добавить поле в словарь.

  2. Написать функцию upcoming_birthdays(days), возвращающую контакты с днями рождения в ближайшие N дней.

Задание 27. Журнал успеваемости студентов#

  1. Изменить add_grade(), добавив параметр weight=1 (вес оценки: контрольная — 2, домашняя работа — 1 и т. д.). Добавить поле в словарь.

  2. Написать функцию weighted_avg_grade(student), возвращающую средневзвешенный балл студента.

Задание 28. Каталог книг в библиотеке (расширенный)#

  1. Изменить add_book(), добавив параметр pages=0. Добавить поле в словарь.

  2. Написать функцию long_books(min_pages), возвращающую книги с количеством страниц больше указанного.

Задание 29. Учёт заказов в кафе#

  1. Изменить add_order(), добавив параметр waiter=None (официант). Добавить поле в словарь.

  2. Написать функцию orders_by_waiter(waiter), возвращающую заказы указанного официанта.

Задание 30. Список задач для волонтёров#

  1. Изменить assign_task(), добавив параметр location=None (место выполнения). Добавить поле в словарь.

  2. Написать функцию tasks_by_location(location), возвращающую задачи для указанного места.

Задание 0. Система учёта оценок студентов#

Структура данных:

students = [
    {'name': 'Анна', 'grades': [4, 5, 4], 'group': 'ФИЗ-101'},
    # ...
]

Функции:

  1. add_student(name, group) — добавляет нового студента (список оценок пустой).

  2. add_grade(name, grade) — добавляет оценку студенту по имени.

  3. average_grade_by_group(group) — возвращает средний балл по группе.

  4. remove_student(name) — удаляет студента из списка по имени.

Доп задание

  1. Изменить функцию add_student так, чтобы она принимала еще один параметр grades, который по умлочнию равен None. Если grades число оценка или список оценков, добавить поле студента.

  2. Добавить функцию add_av_grade_for_st(), которая по добавляет ко всем студентам поле av_grade, в котором харниться средняя оценка студентов.
